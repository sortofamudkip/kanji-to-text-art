# kanji-to-text-art
 
Given a Kanji (漢字), convert it into a text art rendition of itself.

## Example

Input: "字"

Output: 
```
　　　　　　　　　字　　　　　　　　　　
　　　　　　　　　字字　　　　　　　　　
　　字字字字字字字字字字字字字字字字字　
　　字　　　　　　　　　　　　　　字　　
　字字　　　　　　　　　　　　　字　　　
　字　　字字字字字字字字字字字　　　　　
　　　　　　　　　　　　　字字　　　　　
　　　　　　　　　　　字字　　　　　　　
　　　　　　　　　字字　　　　　　　　　
　　　　　　　　　字字　　　　　　　　　
　　　　　　　　　字字　　　　　　字字　
　字字字字字字字字字字字字字字字字字字　
　　　　　　　　　字　　　　　　　　　　
　　　　　　　　　字　　　　　　　　　　
　　　　　　　　　字　　　　　　　　　　
　　　　　　　　　字　　　　　　　　　　
　　　　　　字　　字　　　　　　　　　　
　　　　　　　　字字　　　　　　　　　　
```
